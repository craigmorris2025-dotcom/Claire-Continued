## Canonical Cockpit Surface Registry

Creates a single governed registry for active cockpit surfaces.

Tracks:

- surface ID
- payload ownership
- route ownership
- sidebar ownership
- telemetry ownership
- visibility state
- synchronization state
- runtime authority state

Preserved:

- backend owns truth
- cockpit presentation only
- runtime authority blocked
- fail-closed governance
- no autonomous execution expansion
- no new cockpit generation
