# Claire v17.73 - Automatic Update Runner Gate

This build registers the automatic update runner gate.

It does not execute updates.

## Purpose

The runner gate evaluates whether a staged update pack and rollback plan exist and whether update governance is intact.

## API routes

- `GET /updates/runner-gate`
- `GET /updates/runner-gate/summary`
- `POST /updates/runner-gate/rebuild`
- `GET /updates/automatic/status`

## Output files

- `data/update_packs/automatic_update_runner_gate.json`
- `data/dashboard/automatic_update_runner_gate_payload.json`

## Critical rule

The runner is gate-only in this build.

No background execution. No hidden updates. No uncontrolled self-modification. No bypassing rollback, validation, source trust, or operator review.
