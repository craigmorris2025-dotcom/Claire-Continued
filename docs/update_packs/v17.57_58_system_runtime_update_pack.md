# System Runtime Update Pack — Claire v17.57-v17.58

## Name

Runtime Truth Reader + Route Contract Probe

## Purpose

Add a safe runtime utility that converts existing `core_run_output.json` files into `exports/latest/dashboard_runtime_truth.json` for the dashboard.

## Installed utility

```bash
python tools/claire_runtime_truth_probe.py
```

The utility searches for the newest runtime output and writes a dashboard-readable truth file.

## Output

```text
exports/latest/dashboard_runtime_truth.json
```

## Runtime contract fields normalized

- run_id
- timestamp
- route_selected
- route_decision
- routes_rejected
- terminal_state
- validation_result
- validated_output_path
- memory_eligibility
- evidence_chain
- confidence_scores
- governance_stamp
- lifecycle_stages
- stage_counts
- route_contract
- proof_checks

## Safety

This build does not modify runtime orchestration or import itself into the FastAPI app. It is rollback-safe and non-invasive.
