# Claire v17.75.2 - Workspace Dashboard + Web Search + Agent Command Bar Preparation

This build changes the dashboard from a long stacked report page into a workspace cockpit while preserving the permanent search bar.

## Search bar purpose

The search bar is a permanent core surface. It is prepared for:

- normal web search
- governed research search
- Claire runtime/system search
- future AI-agent command mode

## Safety

Normal web search is prepared but not uncontrolled. Live web calls remain disabled until governed source trust, allowlists/quarantine, evidence capture, rate limits, and operator-visible results are active.

Agent command mode is prepared but autonomous execution remains disabled.

## API routes

- `GET /operator/search/capabilities`
- `POST /operator/search/query`
- `POST /operator/command/parse`
- `GET /operator/search/web/status`
- `POST /operator/search/web`
- `GET /operator/search/layer`
