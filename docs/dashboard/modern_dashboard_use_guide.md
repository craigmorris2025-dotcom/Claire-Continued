# Claire Modern Dashboard Use Guide

This installs a modern command-center web UI for Claire.

## Install

From the Claire root:

```bat
python claire_modern_dashboard_web_ui_installer.py --install
```

## Installed files

```text
src/frontend/command_center/modern/index.html
src/frontend/command_center/modern/modern_dashboard.css
src/frontend/command_center/modern/modern_dashboard.js
docs/dashboard/modern_dashboard_use_guide.md
data/dashboard/render_contracts/modern_dashboard_contract.json
```

## Open

Use your normal desktop launcher first:

```bat
START_CLAIRE_DESKTOP.bat
```

Then open:

```text
src/frontend/command_center/modern/index.html
```

This installer does not modify runtime, routing, scoring, live permissions, or core_run_output.json.
