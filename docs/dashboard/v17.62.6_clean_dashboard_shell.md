# Claire v17.62.6 — Clean Dashboard Shell Rebuild

## Purpose

The v17.62 hotfix chain proved the old dashboard shell was carrying too many conflicting injected layers. v17.62.6 replaces the modern dashboard index with one clean, stable operating shell.

## What it does

- Backs up the existing dashboard.
- Replaces `src/frontend/command_center/modern/index.html`.
- Installs one clean stylesheet.
- Installs one clean dashboard script.
- Removes dependency on old v17.62 overlay/bridge scripts.
- Keeps runtime truth honest.
- Reads the same runtime/validation/evidence/memory files.

## Runtime data sources

The dashboard attempts to read:

```text
exports/latest/dashboard_runtime_truth.json
exports/latest/validation_authority_report.json
exports/latest/evidence_traceability_index.json
exports/latest/verified_memory_gate_report.json
exports/latest/recursive_feedback_gate_report.json
```

and local dashboard status mirrors when available.

## Scope

No backend logic changed.
No runtime logic changed.
No validation logic changed.
No memory logic changed.
This is a dashboard shell rebuild.
