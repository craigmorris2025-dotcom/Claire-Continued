# Claire v17.62.1 — Dashboard Decongestion Hotfix

## Purpose

The v17.62 search/responsive layer made the dashboard too congested on wide desktop views. This hotfix repairs the layout without changing runtime logic.

## Fixes

- Search bar docked to top-right instead of floating across the middle.
- Search dropdown no longer opens an empty message just from focus.
- Primary portals no longer create unnecessary nested scrollbars.
- Portals expand vertically so content is viewable.
- Validation and memory panels get wider internal grids.
- Route/lifecycle/card grids use larger readable minimum widths.
- Side and live rails remain scrollable, but main content breathes.
- Adds a small Focus View toggle to hide rails when the workspace feels crowded.
