# Claire v17.62.4 — Workspace Stack Fix

## Purpose

The dashboard still showed a clipped Command panel and overlapping validation/memory surfaces after v17.62.3. This hotfix adds a small JS classifier and CSS stack fix to force major center workspace surfaces into normal vertical flow.

## What it fixes

- Command panel stops clipping its internal runtime cards
- Command panel scroll container is neutralized
- Validation Authority panel is moved into the center stack if it was mounted inside a cramped surface
- Verified Memory panel is moved into the same stack after validation
- Large status pills stop behaving like giant circular badges
- Main workspace sections flow vertically
