# Claire v17.67 - AutoDesign Handoff Contract

This build creates the contract that moves Claire from route audit into AutoDesign.

## Purpose

If a discovery, breakthrough, innovation, system, solution, or invention route requires a buildable design, Claire must produce an AutoDesign handoff request instead of stopping at classification.

## API routes

- `GET /autodesign/handoff`
- `GET /autodesign/handoff/summary`
- `POST /autodesign/handoff/rebuild`
- `GET /design/autodesign/handoff`

## Output files

- `data/autodesign/autodesign_handoff_contract.json`
- `data/dashboard/autodesign_handoff_payload.json`

## Critical rule

AutoDesign is not decoration. It is the required downstream route for discoveries, breakthroughs, or innovations that require a new technology, product, system, architecture, or buildable solution.

Missing inputs remain missing. The handoff contract does not invent fake AutoDesign output.
