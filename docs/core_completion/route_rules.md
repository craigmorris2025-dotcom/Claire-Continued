# Route Rules

Claire starts from trend discovery and portfolio intelligence. Breakthrough escalation is conditional.

Normal early path:

Signal Governance -> Trend Discovery -> Thesis Formation -> Portfolio Creation / Optimization

Conditional breakthrough path:

Qualified discovery -> Breakthrough Classification -> Advancement Path Selection -> Route-specific execution

Routes:

- `portfolio_only`: portfolio construction and acquisition-readiness can proceed without auto invention.
- `breakthrough_design`: design, invention, buildability, deployability, feasibility, and blueprint outputs become required.

Breakthrough classification must not assume technological invention. A breakthrough may be financial, regulatory, operational, portfolio-driven, market-structure-based, or otherwise structural.
