# v1 Product Alignment

Verified date: 2026-05-24

The product narrative now matches the locked runtime shape.

Claire is not positioned as a loose AI chat layer. It is a governed intelligence runtime that turns weak signals, evidence, route contracts, and lifecycle decisions into reviewable outputs.

The v1.0 product claims are bounded by verified runtime behavior:

- endpoint reconciliation is clean
- dashboard controls resolve to backend-owned routes or explicit cleanup candidates
- ACS2 is the canonical trigger-score-route decision map
- standards controls are attached to real endpoints, tests, gates, and runtime behaviors
- Design Portal output and CAD intent are reviewable
- CAD export is contract-prepared and intentionally disabled
- autonomous live web execution, runtime mutation, and automatic update apply remain owner-gated
