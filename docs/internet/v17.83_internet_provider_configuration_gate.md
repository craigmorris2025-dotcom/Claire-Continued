# Claire v17.83 - Internet Provider Configuration Gate

This build prepares Claire's permanent search bar for normal web search without enabling uncontrolled browsing.

## Purpose

Claire's search bar must eventually support:

- normal web search
- governed research search
- Claire runtime/system search
- future AI-agent command mode

v17.83 creates a provider gate and templates for configuring a search provider safely.

## Routes

- `GET /internet/provider-gate`
- `GET /internet/provider-gate/summary`
- `POST /internet/provider-gate/rebuild`
- `GET /internet/search/provider/status`

## Output files

- `data/internet_provider/provider_configuration_gate.json`
- `data/internet_provider/provider_config_template.json`
- `data/internet_provider/source_allowlist_template.json`
- `data/internet_provider/quarantine_policy.json`
- `data/internet_provider/.env.internet.example`
- `data/internet_provider/v17_83_internet_provider_stop_go.json`
- `data/internet_provider/v17_83_internet_provider_stop_go.md`
- `data/dashboard/internet_provider_gate_payload.json`

## Critical rule

This build does not run web searches and does not enable live internet. It only checks/provider-prepares the configuration gate.
