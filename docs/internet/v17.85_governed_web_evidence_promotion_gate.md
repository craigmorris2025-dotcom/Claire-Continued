# Claire v17.85 - Governed Web Evidence Promotion Gate

This build adds the review gate after a governed live web probe.

## Purpose

Live search results should not automatically become runtime truth. This build creates a manual evidence-promotion layer.

## Routes

- `GET /internet/evidence-promotion`
- `GET /internet/evidence-promotion/summary`
- `POST /internet/evidence-promotion/rebuild`
- `POST /internet/evidence-promotion/promote-approved`
- `GET /internet/evidence-promotion/status`

## Output files

- `data/internet_evidence/evidence_promotion_gate.json`
- `data/internet_evidence/promoted_evidence_candidates_template.json`
- `data/internet_evidence/quarantine_review_queue.json`
- `data/internet_evidence/promoted_evidence_store.json`
- `data/internet_evidence/evidence_promotion_execution_report.json`
- `data/internet_evidence/v17_85_evidence_promotion_stop_go.json`
- `data/internet_evidence/v17_85_evidence_promotion_stop_go.md`
- `data/dashboard/internet_evidence_promotion_payload.json`

## Critical rule

Promotion requires manual review and confirm text. Promoted evidence still does not feed runtime truth until a later runtime-truth ingestion gate.
