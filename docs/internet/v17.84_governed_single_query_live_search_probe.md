# Claire v17.84 - Governed Single-Query Live Search Probe

This build prepares the first real live web search probe while preserving governance.

## Routes

- `GET /internet/live-probe/status`
- `GET /internet/live-probe/contract`
- `POST /internet/live-probe/run`
- `GET /internet/live-probe/last`

## Critical rule

This installer does not run a web search. A live search can only happen from an explicit operator request with confirm text:

`RUN GOVERNED WEB PROBE`

## Safety

- single query only
- operator confirmation required
- source trust required
- unknown domains quarantined
- evidence captured
- results do not feed runtime truth automatically
- automatic updates disabled
- autonomous agent execution disabled
