# Claire v17.66 - Discovery / Breakthrough / Innovation Route Audit

This build audits whether Claire's runtime truth is preserving the full route chain:

signal -> discovery -> thesis -> route gate

Then the route can become:

- portfolio intelligence
- discovery / thesis route
- breakthrough / innovation classification
- AutoDesign handoff
- Design Portal output
- acquisition / package route

## Critical rule

Discovery, breakthrough, and innovation are first-class route outcomes.

If the selected route requires technology invention, system design, product design, architecture, or buildable solution generation, AutoDesign and Design Portal are required surfaces. Missing data is reported as missing, not filled in.
