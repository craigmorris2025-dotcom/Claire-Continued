CLAIRE FORWARD INSTALL RULE

Install scripts are allowed only if they:
1. declare every file they create or replace
2. create backups before replacement
3. avoid generated/test/demo pollution in active runtime
4. write an install manifest
5. run compile validation after install
6. preserve /docs launch
7. do not add placeholder tests to active pytest
8. keep incomplete scaffolds disabled until implemented

Current protected runtime:
- main.py
- src/claire/app.py
- src/claire/api/
- src/claire/orchestrator/
- src/claire/lifecycle/
- src/claire/engines/
- src/claire/output/
- src/claire/dashboard/
- src/claire/technology/technology_intelligence.py

Never move or overwrite protected runtime without backup.
