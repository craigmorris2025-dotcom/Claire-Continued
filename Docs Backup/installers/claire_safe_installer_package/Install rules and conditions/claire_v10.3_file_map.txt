================================================================================
  INSTALLER GOVERNANCE
================================================================================

Each phase follows Claire's existing installer governance pattern:

  1. PREFLIGHT
     ├── Validate target directory exists
     ├── Check no file collisions with existing files
     ├── Verify dependency phases completed
     ├── Generate preflight report → .claire_install/
     └── Write to .claire_install_logs/

  2. INSTALL
     ├── Create new folders
     ├── Write new files (stub templates with docstrings)
     ├── Register each file in install registry
     └── Generate install manifest

  3. POSTFLIGHT
     ├── Verify all files written successfully
     ├── Calculate file checksums
     ├── Update version.json
     ├── Generate postflight report → .claire_install/
     └── Write completion audit → .claire_install_logs/
